6.Write a program to check if the character ‘e’ is present in the given string “Umbrella”.
Ans: public static void main(String[] args){
String a = "Umbrella";
boolean per = false;
for(int i = 0;i<a.length();i++){
if(a.charAt(i) == 'e'){
per=true;
break;
}
}
System.out.println(per);
}
}


Output : true
